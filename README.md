## Démarche pour démarrer le projet

1. Clonez le projet :
    ```bash
    git clone <url-du-projet>
    ```
2. Installez les dépendances :
    ```bash
    npm install
    ```
3. Démarrez le projet :
    ```bash
    npm run start
    ```
